# 🛒 Amazon Clone

A responsive **Amazon Clone** built using **HTML, CSS, and JavaScript**. This project recreates the core shopping experience of Amazon with a clean, responsive, and interactive user interface.

---

## 📌 Overview

This project is a frontend clone of the Amazon website developed to enhance my web development skills. It focuses on building a responsive e-commerce interface using HTML, CSS, and JavaScript while following modern UI design principles.

---

## 🚀 Features

- 🏠 Responsive Home Page
- 🔍 Product Search Bar
- 🛍️ Product Listing
- 📦 Product Details
- 🛒 Shopping Cart
- ➕ Add to Cart Functionality
- ❌ Remove from Cart
- 💰 Order Summary
- 🚚 Delivery Options
- 📱 Responsive Design
- ✨ Interactive User Interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Xavierdivya/amazon-clone-html-css-js.git
```

2. Navigate to the project folder

```bash
cd amazon-clone-html-css-js
```

3. Open `amazon.html` in your browser.
